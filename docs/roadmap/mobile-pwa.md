# Roadmap: Mobile & PWA

## Built

- **PWA Installable** — Manifest, standalone display, home screen icons, theme color.
- **Hamburger Menu** — Slide-out drawer replaces sidebar on mobile.
- **Safe Area Insets** — Top (notch) and bottom (home indicator/address bar) handled.
- **Version Badge** — v0.2.0 in bottom-left on mobile.
- **Camera Scanner** — Shutter button + OCR pipeline (parked: needs API key).

## Ideas

- **Offline Mode** — Cache deck data in IndexedDB for viewing without connectivity
- **Push Notifications** — "Price alert: Mana Crypt dropped 20%" or "Wishlist card acquired"
- **Quick-Add Widget** — iOS/Android home screen widget: tap → camera opens for quick scan
- **NFC Tag Integration** — Tap NFC sticker on deck box → opens that deck in Oracle
- **Deck Box Mode** — Simplified mobile view: just show the decklist + completeness (for at-the-table reference)
- **Share Sheet** — Share deck as text/image from mobile (native share API)
