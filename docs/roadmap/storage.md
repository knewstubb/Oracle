# Roadmap: Storage & Organization

## Built

- **Binders** — Named physical locations (Trade Binder, Commander Staples, Bulk Box).
- **Assign Cards** — Cards exist in one place: a deck OR a binder.
- **View Contents** — Browse cards per binder with search, count, total value.

## Planned

### Proxy Print Sheet
**Priority:** #4 | **Effort:** Medium

Button on deck page: "Print Proxies." Generates printable grid at card dimensions (2.5"×3.5"). 9 per page. Uses Scryfall images. Output: print-optimized HTML page (simplest, no dependencies).

## Ideas

- **QR Code Labels** — Generate QR codes for physical boxes. Scan → see contents.
- **Storage Suggestions** — "47 unassigned green cards — create a Green Staples binder?"
- **Photo Documentation** — Attach photos to binder records. Visual confirmation + insurance.
- **Pull Lists** — "Go to Green Staples binder, pull cards at positions 12, 34, 67" (for deck assembly)
- **Audit Mode** — Walk through a binder card-by-card. "Still here? Yes/No." Bulk verification.
